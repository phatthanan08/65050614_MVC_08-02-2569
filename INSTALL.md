# วิธีการติดตั้งและรันโปรแกรม

## ไลบรารี่ที่ต้องติดตั้ง

ไม่ต้องติดตั้งเพิ่มเติม - ใช้ Python built-in libraries เท่านั้น:
- `tkinter` (มากับ Python อยู่แล้ว)
- `csv` (built-in)
- `os` (built-in)
- `datetime` (built-in)

**ความต้องการ:** Python 3.6 ขึ้นไป

---

## วิธีรันโปรแกรม

### Windows
```bash
python main.py
```

### macOS / Linux
```bash
python3 main.py
```

---

## ตรวจสอบว่า Python พร้อมใช้งาน

```bash
python --version
```

ควรแสดง Python 3.6 หรือสูงกว่า
